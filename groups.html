Community for F#

The Community for F# provides resources to connect you to the F# community. This includes information about existing events and user groups, live streaming and recordings of user group presentations, as well as content for you to use in presenting to your own local user groups.




View F# User Groups in a larger map.

Note that the location of meetings frequently changes for many of the user groups, so always check the user group page for the most up-to-date information. If you are running an F# user group that is missing from this list, please contact @reedcopsey.
